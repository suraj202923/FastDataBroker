🚀 DEPLOYMENT OPTIONS

You can deploy using any of these methods:

1. Local Development
   → cargo run --release

2. Docker
   → docker build -t admin-api . && docker run -p 8080:8080 admin-api

3. Docker Compose
   → docker-compose up -d

4. Kubernetes
   → kubectl apply -f kubernetes/

5. Systemd Service
   → Follow DEPLOYMENT.md for systemd setup

6. Load Balanced (Nginx/HAProxy)
   → See DEPLOYMENT.md for reverse proxy setup
